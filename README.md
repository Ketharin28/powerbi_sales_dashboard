# powerbi_sales_dashboard
Sales Performance Dashboard created using Power BI
📊 Power BI Assignment – Sales Performance Dashboard
📌 Project Overview

This project is a Power BI Sales Performance Dashboard created to analyze business performance across different categories, products, and time periods.

The dashboard provides insights into:

Sales performance

Profit trends

Discount impact

Category-wise analysis

Top 5 products performance

Year-over-Year comparison

The goal of this project is to convert raw business data into meaningful visual insights for decision-making.

🎯 Objectives

Analyze total Sales, Profit, and Discount.

Identify Top 5 performing products.

Compare current year performance with previous year.

Perform category-wise analysis (Furniture, Technology, Office Supplies).

Visualize KPIs using cards, charts, and conditional formatting.

📂 Dataset Information

The dataset contains the following key fields:

Order Date

Product Name

Category

Sales

Profit

Discount

Customer Segment

Region

The data was cleaned and transformed using:

Power Query Editor

Data type formatting

Removal of duplicates (if applicable)
